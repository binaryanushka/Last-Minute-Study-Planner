📚 Last-Minute Study Planner

A smart and simple web application designed to help students manage their study schedule efficiently during exam time.

---

🚀 Features

📖 Subject Management

- Add subjects with exam dates and priority levels (High / Medium / Low)
- View subjects in a clean card layout
- Automatically calculates days left for each exam
- Delete subjects easily
- Data stored using localStorage

---

🧠 Smart Study Planner

- Automatically generates a study schedule
- Prioritizes subjects based on upcoming exam dates
- Distributes study sessions intelligently

---

📅 Today's Tasks

- Displays tasks specifically for the current day
- Mark tasks as completed
- Saves progress locally

---

📊 Progress Tracking

- Dynamic progress bar based on completed tasks
- Helps track study consistency

---

⏳ Pomodoro Timer (Per Subject)

- 25-minute focused study session
- 5-minute break timer
- Start / Pause / Reset controls
- Independent timer for each subject
- Helps improve focus and productivity

---

✨ Additional Features

- Motivational quote section
- Subjects sorted by nearest exam date
- Clean and responsive UI
- Smooth animations and modern design

---

🛠️ Tech Stack

- LocalStorage (for saving data)

This app helps by:

- Creating a smart study plan
- Improving focus using Pomodoro technique
